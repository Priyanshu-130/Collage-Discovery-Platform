# College Discovery Platform

College Discovery Platform is a modern, high-performance web application built with Next.js (App Router) and TypeScript. It helps prospective students research, filter, compare, bookmark, and predict admission chances across India's top colleges and universities.

Built as part of an AI Software Engineer Internship Assignment for the Frontend Engineer role under the College Discovery Platform track.

---

## Features

### Interactive Dashboard
- Dynamic platform statistics
- Featured colleges
- Recently viewed colleges
- Quick search experience

### College Explorer & Advanced Filtering
- Multi-criteria search
- Advanced filters
- Sorting options
- Loading states
- Empty states

### College Detail Profiles
- College overview
- Fees and placements
- Course information
- Campus metrics
- Detailed ratings

### Admission Predictor
- Rank-based prediction
- Branch-level recommendations
- Multiple entrance exam support
- Prediction history

### College Comparison
- Compare up to 4 colleges
- Side-by-side comparison
- Placement and fee analysis
- Saved comparison groups

### Saved Colleges
- Bookmark colleges
- Persistent storage using LocalStorage
- Quick-access saved dashboard

### Authentication Simulator
- Login and signup simulation
- User-specific saved data
- Session management experience
